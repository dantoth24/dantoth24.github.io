Projects

These are some projects that I've worked on over the years. Some of the code cannot be shared due to school restrictions.

Rogue Reckoning

A rouguelike game created in Unity and C#. I implemented a data structure of nodes that procedually generated different configurations while still maintaining a basic structure. I also made the user interface and some of the associated logic.

Text Analyzer

A natural language processing program that responds to inquiries about what takes place in an input text and provides an appropriate response. It uses NLTK sililarity scores to catagorize words. Implemented using Python.

Stereo Vision

A C program that takes in bitmap images from slightly different perspectives and creates a depth map from the changes. Also implemented quadtree compression to store it.

Berkeley Bus Tracker

An android app written in Java that uses the Google Maps API to show bus stops for the lines near UC Berkeley. Made at Hack Davis 2017 with Clara de Martel. WIP.

Happy Bot

A Twitter bot made for Hack Mental Health that checks in on followers after analyzing the sentiment of their tweets. It also posts happy memes sourced from the Reddit API. Made with Noah Alcus and Zachary Yiu.

Gitlet

A light-weight version of Git implemented using Java, with support for most commands including remote repositories and branch merging.
